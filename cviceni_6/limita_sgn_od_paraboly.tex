Spočítejte $\underset{x \rightarrow 0}{\lim} \sgn(x^2)$, kde $\sgn$ je funkce signum, tedy
$$\sgn(x) =
\begin{cases}
	-1 & \text{ pro } x < 0 \\
	0 & \text{ pro } x = 0 \\
	1 & \text{ pro } x > 0
\end{cases}
$$

\solution{
	Pro všechna $x \neq 0$ je $x^2>0$ a tedy $\sgn(x^2)=1$, takže i když je $\sgn(0)=0$, je
	$\underset{x \rightarrow 0}{\lim} \sgn(x^2)=1$
	(limitu zkoumáme na prstencovém okolí).
}
