Rozhodněte, zdali jsou následující posloupnosti $(a_n)_{n = 1}^{\infty}$ monotónní, pokud ano, určete typ monotonie (rostoucí, klesající, nerostoucí, neklesající, konstantní).
\begin{enumerate}
	\item  $a_n = 2n + (-1)^n$
	\item  $\frac{1}{1+n^2}$
	\item  $\frac{n+1}{n+2}$
	\item  $\frac{n+1}{\sqrt{n^2 + 2n - 2}}$
\end{enumerate}
